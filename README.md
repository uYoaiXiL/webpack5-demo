## Installation

```bash
# clone repo
git clonehttps://github.com/uYoaiXiL/webpack5-demo.git

# install deps
yarn
# or
npm i
```

---

## Usage

### Development Server

```bash
yarn start
# or
npm start
```

### Production Bundle

```bash
yarn build
# or
npm run build
```

### Production Bundle PWA

```bash
yarn pwa
# or
npm run pwa
```

### Production Bundle Analyzer

```bash
yarn analyze
# or
npm run analyze
```
